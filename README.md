# DJ Rayvon Website

## Project Description
This project is a professional DJ website for DJ Rayvon. The website includes a home page, about page, media page, gallery page, and booking/contact page.

## How to Run the Website
1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html`.
4. Right-click the file and select **Open with Live Server**.
5. The website will open in the browser.

## Files Included
- `index.html` - Home page
- `about.html` - About page
- `media.html` - Media page
- `gallery.html` - Gallery page
- `contact.html` - Booking form page
- `style.css` - Website styling
- `main.js` - JavaScript functionality

## Features
- Responsive navigation menu
- Media tabs for audio and video
- Gallery filtering by event type
- Booking form validation
- Mobile-friendly design

## Output / Results
The final website displays DJ Rayvon’s services, background, media samples, event gallery, and booking form. Screenshots of the website pages are included in the project report as figures.
